== Description ==
Widget for quick search of air tickets online (avia4you.com.ua)

== Installation ==
1. Unzip the file you downloaded
2. Copy the avia4you_airtickets.php file in your Sidebar Widget folder
3. Enable the plugin in your Plugin administration page.
4. Drag the AVIA4YOU AIR Tickets on your Sidebar
